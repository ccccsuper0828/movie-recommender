## 🔍 问题诊断

### 1. **相似度矩阵尺度不一致**
- **问题**: Content-Based、Metadata-Based、Collaborative 的相似度矩阵可能有不同的数值范围
- **影响**: 直接加权平均时，尺度大的方法会主导结果
- **证据**: Hybrid 表现不如 Collaborative，说明权重分配不合理

### 2. **Content-Based 和 Metadata-Based 表现极差**
- **可能原因**:
  - 文本特征质量低（overview 可能为空或质量差）
  - TF-IDF 参数未优化
  - 相似度计算有误
  - 特征提取不完整

### 3. **Hybrid 权重固定且不合理**
- **当前权重**: (0.3, 0.4, 0.3) - Content, Metadata, CF
- **问题**: Content 和 Metadata 表现很差，但权重却很高
- **应该**: 根据实际性能动态调整权重

### 4. **评估时的用户-物品交互稀疏**
- **问题**: 评估器使用 MovieLens 数据，但可能数据稀疏
- **影响**: 某些用户可能没有足够的测试物品，导致评估不准确

## 🛠️ 优化方案

### 方案 1: 相似度矩阵归一化（高优先级）

**问题**: 不同方法的相似度矩阵尺度不一致

**解决方案**: 在融合前对每个相似度矩阵进行归一化

```python
# 在 hybrid.py 的 _compute_hybrid_similarity() 方法中
def _compute_hybrid_similarity(self):
    """Compute the weighted hybrid similarity matrix with normalization."""
    logger.info("Computing hybrid similarity matrix...")
    
    n_movies = len(self._movies_df)
    self._similarity_matrix = np.zeros((n_movies, n_movies))
    
    # Normalize and add weighted components
    if self._content_recommender.similarity_matrix is not None:
        content_sim = self._content_recommender.similarity_matrix
        # Min-max normalization to [0, 1]
        content_min, content_max = content_sim.min(), content_sim.max()
        if content_max > content_min:
            content_sim_norm = (content_sim - content_min) / (content_max - content_min)
        else:
            content_sim_norm = content_sim
        self._similarity_matrix += self.weights[0] * content_sim_norm
    
    if self._metadata_recommender.similarity_matrix is not None:
        metadata_sim = self._metadata_recommender.similarity_matrix
        metadata_min, metadata_max = metadata_sim.min(), metadata_sim.max()
        if metadata_max > metadata_min:
            metadata_sim_norm = (metadata_sim - metadata_min) / (metadata_max - metadata_min)
        else:
            metadata_sim_norm = metadata_sim
        self._similarity_matrix += self.weights[1] * metadata_sim_norm
    
    if self._cf_recommender.similarity_matrix is not None:
        cf_sim = self._cf_recommender.similarity_matrix
        cf_min, cf_max = cf_sim.min(), cf_sim.max()
        if cf_max > cf_min:
            cf_sim_norm = (cf_sim - cf_min) / (cf_max - cf_min)
        else:
            cf_sim_norm = cf_sim
        self._similarity_matrix += self.weights[2] * cf_sim_norm
```

### 方案 2: 基于性能的动态权重调整（高优先级）

**问题**: 固定权重不合理

**解决方案**: 根据各方法的评估性能动态调整权重

```python
# 在 hybrid.py 中添加方法
def optimize_weights_from_evaluation(
    self,
    evaluator: 'RecommenderEvaluator',
    k: int = 10
) -> Tuple[float, float, float]:
    """
    Optimize weights based on evaluation results.
    
    Returns
    -------
    tuple
        Optimized weights (content, metadata, cf)
    """
    # Evaluate each method
    content_results = evaluator.evaluate(
        self._content_recommender.similarity_matrix,
        "Content-Based",
        k=k
    )
    metadata_results = evaluator.evaluate(
        self._metadata_recommender.similarity_matrix,
        "Metadata-Based",
        k=k
    )
    cf_results = evaluator.evaluate(
        self._cf_recommender.similarity_matrix,
        "Collaborative",
        k=k
    )
    
    # Use NDCG as the performance metric
    content_ndcg = content_results.get('ndcg@k', 0.0)
    metadata_ndcg = metadata_results.get('ndcg@k', 0.0)
    cf_ndcg = cf_results.get('ndcg@k', 0.0)
    
    # Softmax-like weighting (avoid zero weights)
    scores = np.array([content_ndcg, metadata_ndcg, cf_ndcg])
    scores = np.maximum(scores, 0.01)  # Minimum weight
    weights = scores / scores.sum()
    
    self.set_weights(weights[0], weights[1], weights[2])
    return tuple(weights)
```

### 方案 6: 添加相似度矩阵后处理（中优先级）

**问题**: 相似度矩阵可能包含噪声

**解决方案**: 添加后处理步骤

```python
# 在 base_recommender.py 或各推荐器中添加
def _postprocess_similarity(self, similarity_matrix: np.ndarray) -> np.ndarray:
    """
    Post-process similarity matrix:
    1. Set diagonal to 0 (exclude self)
    2. Apply threshold (remove very low similarities)
    3. Normalize rows
    """
    # Set diagonal to 0
    np.fill_diagonal(similarity_matrix, 0)
    
    # Apply threshold (optional)
    # similarity_matrix[similarity_matrix < 0.1] = 0
    
    # Row normalization (L2 norm)
    row_norms = np.linalg.norm(similarity_matrix, axis=1, keepdims=True)
    row_norms[row_norms == 0] = 1  # Avoid division by zero
    similarity_matrix = similarity_matrix / row_norms
    
    return similarity_matrix
```
